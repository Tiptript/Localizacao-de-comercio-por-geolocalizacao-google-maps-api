Extrator de Dados Geográficos - Google Maps APIDescriçãoScript em Python desenvolvido para automatizar a extração de dados de estabelecimentos comerciais utilizando a API do Google Maps. A ferramenta realiza buscas baseadas em palavras-chave dentro de um raio geográfico específico a partir de um endereço base. O script processa os dados de retorno (Nome, Endereço, Telefone, Latitude e Longitude) e gera um arquivo CSV formatado para importação direta no Google My Maps.Pré-requisitosAmbiente de Execução: Python 3.x.Bibliotecas Necessárias: googlemaps, pandas.Credenciais do Google Cloud Console:Chave de API válida.Geocoding API ativada (para conversão do endereço base em coordenadas).Places API ativada (para busca dos estabelecimentos).Instalação de DependênciasExecute o comando abaixo para instalar os pacotes necessários:Bashpip install googlemaps pandas
ConfiguraçãoAntes da execução, é necessário alterar os parâmetros no bloco de configurações do arquivo buscar.py:API_KEY: Insira a chave de API do Google Cloud.ENDERECO: String contendo o endereço que servirá como ponto central (marco zero) para a busca.RAIO_METROS: Raio de abrangência da busca em metros (ex: 5000 para 5 km).PALAVRAS_CHAVE: Lista de strings contendo os termos de busca (ex: categorias de negócios, produtos ou serviços).ExecuçãoInicie o script via linha de comando:Bashpython buscar.py
Estrutura de Saída (Output)O script gera um arquivo nomeado marcenarias_para_mapa.csv no diretório de execução. O uso da vírgula como delimitador padrão e a codificação UTF-8 garantem compatibilidade nativa com a ferramenta de importação do Google My Maps.O arquivo gerado contém as seguintes colunas:ColunaDescriçãoNomeNome comercial do estabelecimento registrado no Google Places.EnderecoEndereço formatado do local.TelefoneNúmero de contato (retorna "Não informado" em caso de ausência).LatitudeCoordenada Y (necessário para georreferenciamento no mapa).LongitudeCoordenada X (necessário para georreferenciamento no mapa).TermoA palavra-chave específica que engatilhou o retorno do local.Tratamento de Exceções IntegradoErro de Geocoding: Valida se o endereço base fornecido pode ser convertido em coordenadas antes de iniciar o loop de busca.Paginação: Implementa pausa de 2 segundos (timeout exigido pela documentação do Google) entre chamadas sequenciais utilizando o page_token para evitar bloqueios por limite de requisições por segundo.Redundância: Utiliza um conjunto (set) de place_id para garantir que estabelecimentos retornados em múltiplas palavras-chave não sejam duplicados no arquivo final.
